# NexFlow - Automatic Traffic Signal Control System 🚦

## About
A smart traffic signal control system that processes 
real-time traffic data to dynamically adjust signal 
timings and reduce congestion.

## Built With
- Python
- JavaScript
- Adaptive Algorithms

## Features
- Real-time traffic data processing
- Adaptive signal timing management
- Reduces road congestion
- Improves transportation efficiency

## Team Project
Developed as Final Year B.Tech Project (2023-2024)
MES College of Engineering & Technology, Kunnukara

## How to Run
Clone the repository and run the main Python file.
